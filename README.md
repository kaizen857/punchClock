# 基于NFC技术的打卡签到器

## 开发环境

MCU:STM32f412RGT6

环境：ArchLinux + VScode + stm32cubeMX + EIDE + Clangd

cubeMX工具链选择STM32CubeIDE

EIDE构建配置：GCC

EIDE烧录配置：shell(st-util)
